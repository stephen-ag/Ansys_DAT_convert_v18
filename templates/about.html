This web app will Convert the DAT file from higher version to v19 version. The criteria used is blocked indentation from 2020 thermal command, to the apdl command bf,xxxx,temp,300.0

Input Requirements: Make sure you have only one imported thermal file in the setup'. if you see any bf before bfblock, comment the statement and run the code later uncomment it. IgnoreBelow commands if human reading

The Input sheet should contain row 2 starting with "Time" as shown in picture and sheet showing "Interpolate_data" where data is placed for which interpolation required. Make sure the row number 2 should contain the all column heading for which interpolation is required

A new excel sheet will be created with the lookup table and the interpolated values and saved as interpolated file.

Select the input excel file using the Choose file button in the home page and submit the execute button, the output file will be saved
